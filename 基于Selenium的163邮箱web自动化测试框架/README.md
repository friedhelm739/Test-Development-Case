# 基于Selenium的163邮箱web自动化测试框架

>使用PageObject思想编程，方便日后的更改与扩展，目前已完成了163邮箱登录页面，添加联系人页面以及写信页面的测试

直接运行run.py文件即可开始测试添加联系人页面以及写信页面，登录页面建议最后测试，因为163可能会认为咱们在盗号登录，会让输验证码，目前本框架还不支持验证码功能，等以后有空了会添加。

测试用例均在Data文件夹中，为保护个人隐私所有的数据均变为XXX，有兴趣的可以自行更改运行。
